A beautiful, cross platform application to edit [Typst](https://typst.app/) documents. This application supports bidirectional syncing between the source files and the compiled document, instant re-rendering via partial incremental updates, fully integrates [Tinymist LSP](https://github.com/Myriad-Dreamin/tinymist), and provides a sleek, easy to use and minimal document editing experience.


Features:
- Scroll sync between the compiled document and the source files.
- Instant re-rendering of only needed pages
- Export as PDF, PNG, & SVG.
- Bibliography support
- Typst package management
- Entirely local and offline
- Instant playground to get started with new documents
- Full outline, and tools like section extraction
- Fully featured file browser
- Support for multiple documents at once
- Editor also renders PDFs, images, and other media for convenience.
- Git integration - interactive diff editor
